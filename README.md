# Мой выбор - ЗОЖ: Цифровой детокс

Демо: https://kusokmedi.github.io/piramidapitanija/

**Простой учебный проект на HTML/CSS/JS** для создания интерактивного плаката о цифровом детоксе и здоровье мозга.

## Что это

Этот проект представляет собой статическую веб-страницу с плакатом, оформленным в стиле здорового образа жизни. Он показывает:
- контраст между вредными привычками использования гаджетов ночью и полезными привычками цифрового детокса;
- важность сна, отдыха и правильного режима;
- чек-лист действий для поддержки здоровья.

## Как работает

- В основе лежит `index.html` - единственная страница проекта.
- Используется Tailwind CSS через CDN для быстрой стилизации.
- Иконки добавлены с помощью `lucide`.
- Кнопка скачивания сохраняет плакат в формате PNG с помощью `html2canvas`.

## Как запустить

1. Открой `index.html` в браузере.
2. Нажми на кнопку в правом нижнем углу, чтобы скачать плакат.

## Структура проекта

- `index.html` - весь код проекта в одном файле.
- `README.md` - описание проекта.

## Технологии

- HTML5
- CSS (Tailwind через CDN)
- JavaScript
- `html2canvas`
- `lucide` иконки

## Особенности

- адаптивное масштабирование плаката под размер экрана;
- плавное появление и современный дизайн;
- скачивание плаката в качестве PNG.

## Цель проекта

Демонстрация визуального оформления информационного постера и простая реализация цифрового детокса для школьной презентации или личного портфолио.
